DOM(Document Object Model)

Modelo de Objetos para Documentos"D.O.M" é uma interface de programação para documentos HTML e XML. Ele respresenta a estrutura de um documento em formato de arvore, permitindo q os devs acessem e manipulem os elementos de uma pagina da web de forma dinamica. Em JavaScript, o DOM é usado para interagir com os elementos de uma pagina web, como alterar no texto de um paragrafo ou add um novo elemento a pagina.
